Rules

Objective

You have to find a secret code.

Outsmart the app with great guesswork.

You must break the secret code in the fewest number of guesses.

Game Play

On seven colors available, the app selects four of them to constitute a secret code.

You have to try to duplicate the exact colors and positions of the hidden Code pegs.

Each guess is made by placing a row of Code pegs on the unit.

After every guess, the app informs you of your progress using the color indicators.

  • Red Indicator means that "There's no such color in the secret code."
  • Yellow Indicator means that "The color is present in the code, but you placed it at a wrong position."

End of the Game

If you do not crack the code in ten attempts, that game is over.
